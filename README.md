Rock, Paper, Scissors Game (Python)

A command-line Rock, Paper, Scissors game where the user plays against the computer. The game provides feedback on who wins each round and allows repeated play until the user decides to quit.

How to Run

Install Python 3.

Run the program:

python rock_paper_scissors.py

How It Works

The user selects rock, paper, or scissors (r, p, s).

The computer randomly chooses its move.

The program displays both choices and determines the winner.

The user can play multiple rounds until they choose to exit.
